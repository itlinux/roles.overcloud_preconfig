= DNS Set for the Overcloud 

In order to deploy the Overcloud we need to set the DNS on our undercloud netowrk. 
The network used is ctlplane. 

== Example Playbook

    - hosts: undercloud
      roles:
         - { role: dns-for-overcloud }
